# 📝 To-Do List App (Tkinter GUI)

This is a simple desktop To-Do List application built with Python and Tkinter.  
It lets you add and remove tasks with a user-friendly graphical interface.

## ✨ Features

- Add new tasks
- Delete selected tasks
- Easy-to-use GUI
